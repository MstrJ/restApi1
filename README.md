## Project presents: 

|URL   |BODY   | METHOD   |
| ------------ | ------------ | ------------ |
| /api/Post  | - | GET  |
| /api/Post/{id}  | id  | GET  |
| /api/Post  |  title, content |  POST |
| /api/Post  |  id, content |  PUT |
| /api/Post/{id}  |  id |  DELETE |

- It's in the Docker
- Sending new posts to MongoDB 
- Produces new posts which send to Kafka
